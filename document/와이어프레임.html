9. 컴포넌트 구성

9.1 공통 컴포넌트

  • Header: 상단 네비게이션 바
  • Footer: 하단 푸터
  • Button: 재사용 가능한 버튼 컴포넌트
  • Input: 재사용 가능한 입력 필드 컴포넌트
  • Modal: 모달 다이얼로그 컴포넌트
  • Loading: 로딩 스피너 컴포넌트

9.2 페이지 컴포넌트

  • LoginPage: 로그인 페이지
  • SignupPage: 회원가입 페이지
  • BoardListPage: 게시글 목록 페이지
  • BoardDetailPage: 게시글 상세 페이지
  • BoardWritePage: 게시글 작성 페이지
  • BoardEditPage: 게시글 수정 페이지

10. 사용자 인터랙션

10.1 버튼 동작

  • 로그인 버튼: 로그인 처리 후 게시판 목록으로 이동
  • 회원가입 버튼: 회원가입 처리 후 자동 로그인 및 게시판 목록으로 이동
  • 글쓰기 버튼: 게시글 작성 페이지로 이동
  • 수정 버튼: 게시글 수정 페이지로 이동 (작성자만 표시)
  • 삭제 버튼: 삭제 확인 후 게시글 삭제 (작성자만 표시)
  • 목록 버튼: 게시글 목록 페이지로 이동

10.2 폼 검증

  • 실시간 입력값 검증
  • 에러 메시지 표시 (입력 필드 하단)
  • 제출 전 최종 검증

10.3 피드백

  • 로딩 중 스피너 표시
  • 성공/실패 토스트 메시지
  • 에러 발생 시 명확한 에러 메시지
