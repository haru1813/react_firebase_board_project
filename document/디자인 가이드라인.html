1. 디자인 철학

  • 단순성: 불필요한 요소를 제거하고 핵심 기능에 집중
  • 일관성: 전체 앱에서 일관된 디자인 언어 사용
  • 접근성: 모든 사용자가 쉽게 사용할 수 있는 인터페이스
  • 반응형: 다양한 화면 크기에 대응하는 디자인

7. 레이아웃 규칙

  • 컨테이너 최대 너비: 1200px (데스크톱)
  • 패딩: 모바일 16px, 태블릿 24px, 데스크톱 40px
  • 그리드 시스템: 12컬럼 그리드 사용
  • 브레이크포인트:
    • 모바일: ~768px
    • 태블릿: 769px ~ 1024px
    • 데스크톱: 1025px 이상

8. 아이콘 가이드

  • 아이콘 라이브러리: React Icons 또는 Material Icons
  • 아이콘 크기: 16px, 20px, 24px, 32px
  • 색상: 텍스트 색상과 일치하거나 Primary Blue 사용

9. 애니메이션 및 전환

속성 값 용도
기본 전환 0.3s ease 호버, 클릭 효과
빠른 전환 0.15s ease 즉각적인 피드백
느린 전환 0.5s ease 페이지 전환

10. 접근성 가이드라인

  • 색상 대비: WCAG AA 기준 준수 (최소 4.5:1)
  • 키보드 네비게이션: 모든 인터랙티브 요소에 키보드 접근 가능
  • 포커스 표시: 명확한 포커스 인디케이터 제공
  • ARIA 레이블: 스크린 리더를 위한 적절한 레이블 사용
  • 폰트 크기: 최소 14px 이상 유지
